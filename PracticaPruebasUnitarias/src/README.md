## Cómo ejecutar los tests

Para ejecutar las pruebas unitarias de este proyecto, utiliza el siguiente comando de Maven:

```bash
mvn test
```

Este comando compilará el proyecto y ejecutará todas las pruebas definidas en la clase `SmartWalletTest`. Asegúrate de tener Maven instalado y configurado correctamente en tu entorno de desarrollo para ejecutar este comando sin problemas.
